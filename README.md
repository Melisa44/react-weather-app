# react-weather-app
  Open Weather API
## API Kullanımı

Bu proje, Open Weather API'sini kullanır. API'ye erişmek için bir API anahtarı gereklidir.

API Bilgileri:
- Method: GET
- URL: `https://api.openweathermap.org/data/2.5/weather?q={ŞEHİR_ADı}&appid={API_ANAHTARI}`

## Kullanılan Kütüphaneler

- react-scripts
- axios
